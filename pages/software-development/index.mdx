# Form Builder Introduction

For Form Builder we are using RJSF (React JSON Schema Form) which is a JSON Schema Form library for React. It is a simple and easy-to-use form builder that allows you to create forms with JSON Schema. It is highly customizable and provides a lot of features. You can create forms with a variety of input fields, buttons, and more. It is a great tool for building forms quickly and easily.
